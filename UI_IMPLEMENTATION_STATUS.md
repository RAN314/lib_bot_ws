- **解决方案**：需要在UI启动时自动启动RFID传感器节点

- **问题描述**：RFID传感器节点、UI、仿真环境之间的启动依赖关系不明确

- **解决方案**：创建统一的启动脚本或launch文件

- **影响**：控制台输出过于冗杂
