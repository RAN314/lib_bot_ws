## 🚀 当前可用启动方式

### 推荐方式：简单RFID演示
```bash
ros2 launch libbot_simulation simple_rfid_demo.launch.py
```

### 监控RFID检测
```bash
# 查看前方RFID检测
ros2 topic echo /rfid/scan/front

# 查看所有RFID话题
ros2 topic list | grep rfid

# 查看可视化数据
ros2 topic echo /rfid_visualization
```

### 控制机器人
```bash
# 键盘控制
ros2 run teleop_twist_keyboard teleop_twist_keyboard
```
